Introduction To Matrices

A matrix (plural: matrices) is an array of numbers (and/or algebraic unknowns) presented in

brackets.Common usages of matrices in real life includes: Statistics, Computer graphics, Computer Engineering, Robotics, Mechanical engineering.

For matrices with different orders or dimensions, or certain special rules on its elements, there

exist special names, such as:

a) Column Matrix (or Column Vectors)

b) Row Matrix (or Row Vectors)

c) Square Matrix

d) Zero Matrix (or null matrix)

e) Identity Matrix

f) Diagonal Matrix

If two matrices have exactly the same elements (and hence the same order), they are

known as equal matrices.

Addition and Subtraction of Matrices

Only matrices of the same order (or dimension) can be added or subtracted to each other. We

simply add or subtract the elements at the same position.

Scalar Multiplication of Matrices

Multiplying a matrix with a number or unknown is known as scalar multiplication of matrices.

Every element in the matrix is multiplied by the scalar term.

Multiplication of Matrices

In matrix multiplication, the elements in the rows of the matrix on the left are multiplied to

the elements in the columns of the matrix on the right.

When a m × n matrix is multiplied to a n × p matrix, the resultant matrix will have the order m x p

Problem-Solving Using Matrices

A real-life application of matrices is its usage in complex statistical calculations. Typically,

solving such problems using matrices involves the following processes:

a. Representing the given data in the form of a matrix/matrices

b. Combining two or more matrices into one matrix using addition, subtraction, scalar

multiples and matrix multiplication

c. Determine what the resultant matrix represents

d. Sometimes, (b) and (c) are repeated on the resultant matrix from the first combination.

e. Relate the resultant matric to the context of the question.
